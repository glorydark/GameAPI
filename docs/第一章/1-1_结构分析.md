# 1.0 结构分析
# 核心结构
```
|-- gameapi
|   |-- annotation # 用于核心内部标注某些方法是否为测试版/未完成
|   |-- commands # 核心自带指令
|   |-- entity # 用于核心内部的部分实体操作，如排行榜实体
|   |-- event # 小游戏过程中会触发的事件
|   |   |-- block # 方块事件
|   |   |-- entity # 实体事件
|   |   |-- player # 玩家事件
|   |   |-- room # 房间事件，TickEvent每1秒触发一次，非TickEvent仅刚进入该房间状态才执行一次
|   |-- extensions # 一些方便开发者使用的额外功能
|   |   |-- checkPoint # 检测点
|   |   |-- gameLevel # 等级系统（未正式投入使用）
|   |   |-- locker # 粒子、音效等道具（未正式投入使用）
|   |   |-- supplyChest # 补给箱（未正式投入使用）
|   |   |-- task # 任务系统（未正式投入使用）
|   |-- form # 此方法参照了lt-name的GameCore内部的AdvancedForm
|   |-- listener # 房间监听器内部实现部分
|   |   |-- base
|   |   |   |-- GameListenerRegistry # 用于注册房间监听器  
|   |   |   |-- ...
|   |-- manager # 一些必要功能的管理类
|   |   |-- data
|   |   |   |-- PlayerGameDataManager # 玩家对局数据类，可保存/读取相应数据
|   |   |-- extensions 
|   |   |   |-- AdvancedBlockManager # 用于实现一些方块效果，如踩方块给药水效果等
|   |   |   |-- GameLevelSystemManager # 游戏等级（未投入正式使用）
|   |   |   |-- GameTaskManager # 用于与extensions/task/（未投入正式使用）
|   |   |   |-- LockerManager # 粒子、音效管理（未投入正式使用）
|   |   |-- room
|   |   |   |-- CheckpointManager # 检查点管理（Room.class有注册方法）
|   |   |   |-- RoomVirtualHealthManager # 房间虚拟血量（Room.class可进行操作，需要在房间规则设置virtualHealth为true）
|   |   |-- tools
|   |   |   |-- BossBarManager # bossbar管理
|   |   |   |-- GameEntityManager # 内部类，用于生成排行榜、浮空字等
|   |   |   |-- PlayerTempStateManager # 用于储存玩家原来的背包数据，需要在config.yml开启
|   |   |   |-- ScoreboardManager # 计分板管理类
|   |-- ranking # 排行榜类，具体可以在config.yml通过设置使用
|   |-- room
|   |   |-- edit # 模仿lt-name的方便房间编辑流程（未完工）
|   |   |-- executor # 执行器，方便修改相应的默认房间流程
|   |   |   |-- BaseRoomExecutor # 默认执行器
|   |   |   |-- ...
|   |   |-- items
|   |   |   |-- RoomItemBase # 用于游戏前中后的房间道具，可处理交互事件
|   |   |-- teams
|   |   |   |-- BaseTeam # 房间队伍
|   |   |-- Room.java # 房间类
|   |   |-- RoomChatData.java # 房间聊天数据类
|   |   |-- RoomEdit # 房间编辑类（未完工）
|   |   |-- RoomRule.java # 房间规则类
|   |   |-- RoomStatus.java # 房间状态类
|   |   |-- RoomUpdateTask # 每0.15秒（3 tick）执行任务内容，方便进行更加精细的操作
|   |-- task # 内部类，处理房间状态变化、时间流动
|   |-- tools # 一些方便工具类
|   |   |-- AnimationTools # 动作
|   |   |-- BlockTools # 方块
|   |   |-- EntityTools # 实体
|   |   |-- FileTools # 文件
|   |   |-- FireworkTools # 烟花
|   |   |-- FormulaCalculatorTools # 计算简单的公式，如(1+(2-1))
|   |   |-- GhostyTools # 录像类
|   |   |-- ItemTools # 物品类，将物品保存为字符串或从字符串读取（base64）
|   |   |-- SpatialTools # 处理一些坐标、axisbb分割
|   |   |-- TipsTools # 用于解决一些tips计分板等冲突
|   |   |-- WorldTools # 世界工具，用于核心内部加载、卸载地图等
|   |-- utils # 一些有意思的工具类，如json读取、effect简单创建、多于眼泪、计时、数组区间等
|   |   |-- protocol # 一些核心没有的数据包
|   |-- GameAPI.java # 本小游戏核心主类
```
